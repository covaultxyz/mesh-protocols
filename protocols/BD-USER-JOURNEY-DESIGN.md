## Conversation Design Principles

### When to use BUTTONS:
- Clear, finite choices
- Actions that need to happen
- Navigation between modes

### When to use CONVERSATION:
- Clarifying ambiguous requests
- Exploring options
- Emotional/nuanced situations
- Context-dependent responses

### When to use DASHBOARD:
- Data overview
- Multiple items to scan
- Historical view
- Reporting
